> *This program is built with **Node.js** and **Express**. It allows users to fetch information from a website by using an API that gets live information from: [Dragon Ball API](https://dragonball-api.com/). It's an enjoyable program that helps viewers to get to know the vast characters of Dragon Ball Z.*

## 🚀 Features

* **Character Lookup:** Instantly lookup characters by their ID (ranging from 1 to 78).
* **Detailed Stats:** View character images along with their Ki, Max Ki, Race, Gender, and full descriptions.
